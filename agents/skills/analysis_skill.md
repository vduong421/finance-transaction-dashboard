# Finance Analysis Skill

## When Used

Use when the user asks about spending risk, budget review, categories, merchants, trends, or large transactions.

## Input

- finance summary
- category totals
- monthly totals
- top merchants
- top transactions

## Expected Output

- answer
- evidence
- next action
- recommendation
- decision
- risks
- operator actions

## Rules

- Rank categories and merchants using deterministic totals.
- Highlight large transactions separately from category trends.
- Recommend concrete budget review actions.
